## 功能特性

- 自动检测系统架构和内存
- 动态调整网络参数
- 交互式菜单界面
- 支持修改端口、用户名、密码
- 100GB 大文件传输优化
- 配置备份/恢复
- 多服务器管理
- 监控面板

## 快速开始

```bash
git clone https://github.com/Ac-All-Sh/immich-frps.git
cd immich-frps
bash deploy_frps.sh
```
