frps 部署脚本 - 使用文档

GitHub

简介

frps 一键部署脚本是一个交互式工具，用于在 Linux 服务器上快速部署和管理 frps (Fast Reverse Proxy) 服务。

系统要求

  • 操作系统: Debian 10/11/12, Ubuntu 20.04/22.04/24.04
  • 权限: root 用户
  • 依赖: curl, tar (脚本自动安装)

快速开始

git clone https://github.com/Ac-All-Sh/immich-frps.git
cd immich-frps
bash deploy_frps.sh

主菜单说明

选项功能说明
[1]一键部署使用当前配置部署 frps 服务端
[2]修改配置修改端口、用户名、密码、Token 等
[3]查看配置查看当前所有配置项
[4]管理服务启动/停止/重启/状态/日志
[5]生成 frpc 配置生成客户端配置文件
[6]配置备份/恢复导出或导入 frps 配置
[7]多服务器管理管理多个 frps 服务器实例
[8]监控面板查看连接数、流量等统计
[9]卸载 frps清除所有 frps 相关配置

Docker 部署

使用 Docker

# 拉取镜像
docker pull acallsh/frps:0.69.1

# 运行容器
docker run -d \
  --name frps \
  --restart unless-stopped \
  --network host \
  -v /etc/frp/frps.toml:/etc/frp/frps.toml \
  -v /var/log/frp:/var/log/frp \
  acallsh/frps:0.69.1

使用 Docker Compose

version: '3'
services:
  frps:
    image: acallsh/frps:0.69.1
    container_name: frps
    restart: unless-stopped
    network_mode: host
    volumes:
      - /etc/frp/frps.toml:/etc/frp/frps.toml
      - /var/log/frp:/var/log/frp

管理命令

systemctl start frps      # 启动
systemctl stop frps       # 停止
systemctl restart frps    # 重启
systemctl status frps     # 状态
journalctl -u frps -f     # 日志

frpc 客户端配置

serverAddr = "你的服务器IP"
serverPort = 7000
auth.method = "token"
auth.token = "immich_secure_token_2026"
transport.poolCount = 10
transport.tcpMux = true
transport.heartbeatInterval = 10
transport.heartbeatTimeout = 300

[[proxies]]
name = "ssh"
type = "tcp"
localIP = "127.0.0.1"
localPort = 22
remotePort = 6000
注意: 修改配置后需要重启 frps 服务才能生效。

Introduction

frps One-Click Deploy Script is an interactive tool for quickly deploying and managing frps (Fast Reverse Proxy) service on Linux servers.

System Requirements

  • OS: Debian 10/11/12, Ubuntu 20.04/22.04/24.04
  • Privileges: root user
  • Dependencies: curl, tar (auto-installed)

Quick Start

git clone https://github.com/Ac-All-Sh/immich-frps.git
cd immich-frps
bash deploy_frps.sh

[2]Modify ConfigModify ports, username, password, token
[3]View ConfigView all config items
